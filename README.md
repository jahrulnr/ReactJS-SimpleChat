## SimpleChat

Aplikasi ini dibuat menggunakan ReactJS

## Pengujian

- Nodejs v18.16.0
- NPM v9.5.1

## Instalasi

```sh
npm install
cp .env.example .env # edit .env

npm start
```

## Fitur yang telah dibuat

- Login
- Cari Teman (Tambah teman & batalkan request tambah teman)
- Daftar Teman
- Daftar Pesan
- Chat Room
